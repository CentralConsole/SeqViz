# Sequence Viewer

A React component for visualizing genomic data (GenBank) with linear, circular, and detailed views.

- **English**: [Readme.en_us.md](./Readme.en_us.md)
- **简体中文**: [Readme.zh_cn.md](./Readme.zh_cn.md)

## Quick links

- Install: `npm install sequence-viewer react react-dom d3`
- Build library: `npm run build`
- Run demo locally: `npm run dev`
- Deploy demo (Netlify): `npm run build:demo` → publish `dist-demo`
